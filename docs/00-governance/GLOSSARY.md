# Маяк Авито — глоссарий

**Версия:** 1.0  
**Статус:** APPROVED

| Термин | Точное значение |
|---|---|
| `account_id` | Неизменный внутренний UUID аккаунта. Единственная универсальная сущность, связывающая Telegram, MAX и будущий сайт. Не телефон, не email и не Маяк. |
| Account identity | Привязка внешнего provider identity (например Telegram user ID или MAX user ID) к одному `account_id`. Автоматическая склейка по имени, username, аватару или телефону запрещена. |
| Маяк / Beacon | Отдельное пользовательское отслеживание поиска Avito. Не учётная запись и не идентификатор личности. |
| География Маяка | Один город либо одна нативная country-wide/общая выдача, реально поддержанная исходной ссылкой Avito. Несколько городов в одном Маяке запрещены. |
| `source_url` | Исходная готовая ссылка поиска Avito. Сохраняется неизменной. |
| `extracted_snapshot` | Снимок реально извлечённых параметров, контекста и метаданных выдачи Avito. |
| `overrides` | Структурированные пользовательские изменения: удалённые фильтры, цена и будущие изменения builder-а. Не строковая подстановка URL. |
| Effective configuration | Версионированная рабочая конфигурация, детерминированно вычисленная из `extracted_snapshot + overrides`. |
| Baseline | Первый успешный scan Маяка, который фиксирует текущие объявления без массовой рассылки. |
| Listing observation | Нормализованное наблюдение объявления во время конкретного run. |
| Listing state | История объявления, изолированная по Маяку. Для текущей модели уникальная семантика события опирается на `beacon_id + avito_listing_id + normalized_price`. |
| Notification event | Внутреннее событие о новом listing ID или ранее не встреченной для Маяка паре ID+цена. |
| Outbox | Устойчивая очередь намерений доставки, отделяющая создание события от отправки в Telegram/MAX. |
| Delivery | Одна попытка или результат доставки одного notification event через конкретный endpoint/channel. |
| Route | Заменяемый инфраструктурный маршрут доступа к Avito: DirectRoute, HomeWindowsRoute, ServerProxyRoute или MobileProxyRoute. |
| Route lease | Ограниченное по времени назначение маршрута конкретному run. |
| Windows Egress Agent | Домашний Windows-компонент, который поддерживает исходящий защищённый туннель и используется только как route; он не хранит primary DB, секреты, scheduler или queues. |
| CLI-исполнитель | Codex или другой CLI, выполняющий точную техническую задачу ChatGPT без самостоятельных решений. |
| ChatGPT | Разработчик, архитектор и руководитель проекта; принимает технические решения и проверяет работу CLI. |
| `TEMPORARY_MITIGATION` | Явно ограниченная временная защита от немедленного ущерба; не является постоянным исправлением и требует отдельной задачи на корневую причину. |
